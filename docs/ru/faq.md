# Часто Задаваемые Вопросы (FAQ)

## Общие Вопросы

### Что такое Bot-to-Bot Telegram?

**Bot-to-Bot Telegram** — это платформа для создания мультиагентных AI систем в Telegram. Платформа позволяет создавать AI-ботов с Managed Bots, межботовой коммуникацией и интеграцией с AI через MCP (Model Context Protocol).

### Что такое Managed Bots?

Managed Bots — это функция в **Telegram Bot API 9.6**, которая позволяет вашему боту программно создавать и управлять другими ботами. Пользователи могут создавать своих ботов через специальные ссылки без использования @BotFather напрямую.

### Что такое межботовая коммуникация?

Межботовая коммуникация (Bot-to-Bot Communication) позволяет ботам взаимодействовать друг с другом в:
- **Групповых чатах** — через упоминания команд или ответы
- **Business Accounts** — через подключенные бизнес-аккаунты

Как минимум один бот должен иметь включенный режим Bot-to-Bot.

## Технические Вопросы

### Как включить Managed Bots?

1. Откройте @BotFather
2. Выберите вашего бота
3. Перейдите в **Настройки бота** → **Режим управления ботами**
4. Включите "Режим управления ботами"

### Как включить межботовую коммуникацию?

1. Откройте @BotFather
2. Выберите вашего бота
3. Перейдите в **Настройки бота** → **Bot-to-Bot коммуникация**
4. Включите "Режим Bot-to-Bot коммуникации"

### Какие языки и фреймворки поддерживаются?

| Язык | Фреймворк | Статус |
|------|-----------|--------|
| JavaScript/TypeScript | Telegraf | ✅ Поддерживается |
| JavaScript/TypeScript | grammY | ✅ Поддерживается |
| Python | aiogram | ✅ Поддерживается |
| Python | python-telegram-bot | ✅ Поддерживается |

### Что такое MCP (Model Context Protocol)?

MCP — это открытый стандарт для подключения AI моделей к инструментам и сервисам. Он позволяет вашим Telegram ботам взаимодействовать с AI моделями как Claude, GPT и другими.

## Цены и Ограничения

### Какие есть ограничения скорости?

Telegram Bot API имеет следующие ограничения:
- **30 сообщений в секунду** на чат
- **Глобальные ограничения** — варьируются в зависимости от метода
- **429 ошибки** указывают на превышение лимита

Смотрите наше руководство по [Высоким нагрузкам](/ru/high-load) для оптимизации.

### Платформа бесплатна?

Платформа имеет открытый исходный код (лицензия MIT). Однако:
- Вам нужен токен Telegram бота (бесплатно от @BotFather)
- Стоимость хостинга зависит от вашей инфраструктуры
- AI провайдеры (OpenAI, Claude) имеют свою ценовую политику

## Устранение Проблем

### Мой бот не получает сообщения от других ботов

Проверьте эти настройки:
1. Режим Bot-to-Bot включен в @BotFather
2. Отправляющий бот упомянул вашего бота или ответил на его сообщение
3. Ваш бот имеет права администратора ИЛИ отключенную групповую конфиденциальность

### Я получаю ошибки 429

Это ограничение скорости. Реализуйте:
- Алгоритм токен-корзины
- Экспоненциальную задержку
- Очереди сообщений

Подробнее в [Высокие нагрузки](/ru/high-load).

### Создание Managed бота не работает

Убедитесь, что:
1. Режим управления ботами включен
2. Формат ссылки правильный: `https://t.me/newbot/{username}/{bot_username}?name={bot_name}`
3. Username доступен (не занят)

## Безопасность

### Насколько безопасна платформа?

Меры безопасности включают:
- Шифрование токенов ботов при хранении (AES-256-GCM)
- Валидация HMAC-SHA256 для аутентификации пользователей
- Ограничение частоты для предотвращения злоупотреблений
- Защита от бесконечных циклов для межботовой коммуникации

### Как защититься от бесконечных циклов?

Реализуйте следующие меры защиты:
1. **Дедупликация** — кэширование недавних сообщений
2. **Ограничение частоты** — макс. сообщений в секунду
3. **Ограничение глубины** — макс. глубина взаимодействия
4. **Таймауты** — ограничения времени разговора
