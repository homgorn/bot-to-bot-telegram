## Описание

**Bot-to-Bot Telegram** — это комплексная платформа для создания и управления мультиагентными AI системами в Telegram. Платформа позволяет разработчикам создавать сложные системы межботовой коммуникации, оркестрировать несколько AI агентов, интегрироваться с Model Context Protocol (MCP) и создавать масштабируемые высоконагруженные приложения.

Ключевые возможности:
- **Managed Bots** - Программное создание и управление ботами Telegram
- **Bot-to-Bot Communication** - Межботовое общение в группах и через Business Accounts
- **Multi-Agent Orchestration** - Координация нескольких AI агентов для сложных задач
- **MCP Integration** - Подключение AI моделей через Model Context Protocol
- **Telegram Mini Apps** - Создание полнофункциональных веб-приложений в Telegram
- **High-Load Architecture** - Масштабирование для обработки миллионов сообщений

## Основные функции

### Управляемые боты (Managed Bots)

- **Программное создание ботов** - Создание ботов через API без ручного взаимодействия с BotFather
- **Управление жизненным циклом** - Полный контроль над созданием, настройкой и удалением ботов
- **Управление токенами** - Безопасное хранение и получение токенов ботов через `getManagedBotToken`
- **Создание ботов пользователями** - Пользователи могут создавать персональных AI агентов, бизнес-боты, игры
- **Шаблоны ботов** - Предварительно настроенные шаблоны для распространенных сценариев

### Межботовая коммуникация (Bot-to-Bot)

- **Коммуникация в группах** - Боты могут взаимодействовать в одной группе через команды и ответы
- **Режим администратора** - Получение всех сообщений от других ботов с правами администратора
- **Управление конфиденциальностью** - Отключение группового режима конфиденциальности для получения всех сообщений
- **Интеграция с Business Accounts** - Отправка сообщений между ботами через подключенные бизнес-аккаунты
- **Защита от бесконечных циклов** - Реализация мер защиты против бесконечных взаимодействий

### Оркестрация мультиагентных систем

- **Координация агентов** - Оркестрирование нескольких AI агентов, работающих вместе
- **Делегирование задач** - Разделение сложных задач между специализированными агентами
- **Обмен состоянием** - Обмен контекстом и состоянием между агентами
- **Агенты на основе ролей** - Определение агентов с конкретными обязанностями
- **Иерархическая структура** - Создание иерархий агентов для сложных рабочих процессов

### Интеграция MCP

- **Model Context Protocol** - Подключение к AI моделям через стандарт MCP
- **Обнаружение инструментов** - Автоматическое обнаружение доступных инструментов и возможностей
- **Доступ к ресурсам** - Предоставление AI агентам доступа к ресурсам и данным
- **Шаблоны промптов** - Готовые промпты для распространенных рабочих процессов
- **События в реальном времени** - Отправка событий Telegram AI агентам через MCP

### Telegram Mini Apps

- **Полноэкранные приложения** - Создание иммерсивных полноэкранных приложений
- **Нативная интеграция** - Доступ к Telegram SDK для нативных функций
- **Безопасная аутентификация** - Валидация пользователя через initData HMAC-SHA256
- **Платежи** - Интеграция платежей Telegram со Stripe
- **Интеграция с темой** - Автоматическое соответствие теме Telegram

### Высоконагруженная архитектура

- **Горизонтальное масштабирование** - Масштабирование между несколькими экземплярами
- **Ограничение частоты** - Реализация токен-корзины для лимита 30 сообщений/сек
- **Управление очередями** - Обработка всплесков сообщений через очереди
- **Логика повторных попыток** - Экспоненциальная задержка с джиттером
- **Мониторинг здоровья** - Мониторинг состояния системы в реальном времени

## Быстрый старт

### Предварительные требования

- Node.js 18+ или Python 3.10+
- Telegram Bot Token (от @BotFather)
- Docker (опционально, для продакшена)
- PostgreSQL 15+ (для продакшена)
- Redis 7+ (для продакшена)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/homgorn/bot-to-bot-telegram.git
cd bot-to-bot-telegram

# Установка зависимостей
npm install
# или для Python
pip install -r requirements.txt

# Настройка окружения
cp .env.example .env
```

### Запуск

```bash
npm run dev
# или для Python
python main.py
```

## Примеры кода

### Создание управляемого бота

```javascript
// Обработка обновления managed_bot
bot.on('managed_bot', async (ctx) => {
  const { managed_bot, user } = ctx.update.managed_bot;
  
  // Получение токена бота
  const response = await ctx.telegram.callApi('getManagedBotToken', {
    managed_bot_id: managed_bot.id
  });
  
  const token = response.token;
  
  // Создание нового бота
  const newBot = new Telegraf(token);
  newBot.command('start', (ctx) => ctx.reply('Добро пожаловать!'));
  newBot.launch();
});
```

### Оркестрация агентов

```javascript
const agents = {
  'researcher': new ResearchAgent(),
  'analyzer': new AnalyzerAgent(),
  'reporter': new ReporterAgent()
};

async function handleComplexTask(request) {
  const research = await agents.researcher.search(request);
  const analysis = await agents.analyzer.analyze(research);
  return await agents.reporter.format(analysis);
}
```

## Сценарии использования

### Платформа AI агентов

Создайте платформу, где пользователи могут создавать собственных AI агентов с пользовательскими промптами и возможностями.

### Маркетплейс бизнес-ботов

Постройте маркетплейс, где пользователи могут просматривать и устанавливать готовые боты для различных нужд (поддержка клиентов, продажи, продуктивность, игры).

### Рабочие процессы мультиагентных систем

Оркестрируйте несколько специализированных AI агентов, работающих вместе над сложными задачами.

### Интерактивные игры

Создавайте игры в Telegram, где несколько ботов-агентов взаимодействуют с игроками, управляют состоянием игры и предоставляют динамический опыт.

### Инструменты продуктивности

Создавайте продуктивные наборы с несколькими агентами для управления календарем, задачами, заметками и интеграцией с другими сервисами.
